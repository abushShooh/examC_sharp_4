1.
Интерфейс-это набор методов котороые класс обязан использовать он показывает что должен испольнять класс 

Абстрактный класс-это класс который остается и может хранить внутри себя элементы но через интерфейс мы можем использовать их

Итог: различия- абстрактный класс имеет возможность к изменению так как это контракт и методы а интерфейс это только контракт
2.
Статический класс-это класс благодяра которому мы можем призывать какой либо класс Именем не призваивая ни к какому значению
3.
Generics-это тип написания класса нужен для того чтобы облегчить написания кода для работы с разными типами данных и не переписывать этот код для изменения каких либо решений 
4.
Enum-это специальный тип который позволяет хранить не изменяемые набор значений например:Дни недели,месяц,часы,цвет
